## MarkDown syntax

1.标题使用不同数量的"#"来标识是什么层级，可以对应于HTML里面的H1-H6   
2.Alt text   
3.强调 : 左边一个下划线+文字+右边一个下划线 | *号的数量表示强调的级别   
4.换行：两个空格以上+回车   
5.分割线：3个或以上的短横线、星号或者下划线
